Step-by-Step: Host a Static Website with GitHub Pages
Step 1:Create an index.html File
Step 2: Create a script.js File
Step 3: Create a style.css file 

Push to a New GitHub Repo
1.git init
2.git remote add origin  https://github.com/Vaibhav099-pattar/TASK-6-Host-a-Static-Website-with-GitHub-Pages.git
3. git add .
4. git commit -m "My snake and ladder project commit-1"
5. git push origin master

Enable GitHub Pages
1.Go to your repo on GitHub.
2.Click Settings > Pages (or Pages tab).
3.Under Source, select:
    Branch: main
    Folder: / (root)
4.Click Save.
5.Get Your Live Website Link: https://vaibhav099-pattar.github.io/TASK-6-Host-a-Static-Website-with-GitHub-Pages/
